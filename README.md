# recruit_system
### `介绍`
毕设项目: 校企招聘   答辩结束,放出来造福群众

### `软件架构`
采用前后端分离技术
前端使用vue + ElementUI 
后端: ssm 框架以及 springboot技术栈
持久层: MySql 8
Redis 实现定时任务
使用Websocket 实现前端实时通讯


#### `安装教程`

1.  前端(recruit-web):
        npm install
        npm run eval
2.  后端(recruit-sys):
        导入依赖库
        启动数据库环境:数据库版本 8.0+
        启动redis服务
